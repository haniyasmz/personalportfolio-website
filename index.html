    PROJECTS I HAVE WORKED ON

    Growball Cinemax Website Reimagined


    This UI/UX website was designed by my Team and I for our Group Assignment for the Subject KT34503 Human Computer Interaction. This Interface design aims to improve the existing Growball Cinemax Website . The weakness of the existing website is studied, interviews and questionnaire are done to ask users what they would want to improve in a new website design. The new interface design includes a Movie Booking System, Food Ordering System, and a Karaoke Booking System which are not available in the existing website.

    The redesigned website interface aims to provide users with effortless access, ensuring a user-friendly experience even for first-time visitors, minimizing the learning curve.

    The website interface is designed using Figma, a famous collaborative web application for interface design.

    HUAWEI Mobile Service App: Nearby Search App


    This app was developed when I attended the HUAWEI Mobile Service Foundation Course. This app was created using Android Studio and the programming language Java. This application will help user find the nearest place that they would want to visit.

    It is suitable for traveling users who are not familiar with the place that they are visiting. The application can gelp users find the post office, nearby restaurant, the nearest restroom on a long drive, and the nearest cinema.
